.. py:function:: clean_image(pil_image: PIL.Image.Image, white_point: int = 226) -> PIL.Image.Image

   Applies a linear levels adjustment to the image, setting the white point.

   This effectively removes light gray bleed-through by mapping the
   white_point (default 226) and everything above it to pure white (255).

   :param pil_image: The input image.
   :type pil_image: PIL.Image
   :param white_point: The pixel value (0-255) to map to pure white (255).
   :type white_point: int

   :returns: The cleaned image.
   :rtype: PIL.Image
