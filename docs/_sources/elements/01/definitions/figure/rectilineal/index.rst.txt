.. _rectilineal figures:

rectilineal figures
===================

.. index:: figures

.. 

  **Rectilineal figures** are those which are contained by :ref:`segments <segment>`,

  **trilateral** figures being those contained by three, 
  
  **quadrilateral** those contained by four, 
  
  and **multilateral** those contained by more than four straight lines.

  -- Euclid
